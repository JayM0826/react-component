    public TransitionShow(props: object): PropTypes.element source 

    import TransitionShow from 'daifee-react-component/src/TransitionShow.js'

    TransitionShow 使子组件实现简单的“显示和隐藏”过渡动画

    Params:

    NameTypeAttributeDescription
    props object

    传入组件的参数
    props.show boolean
    • default: false

    是否显示子组件（children）
    props.transitionName string

    实现过渡动画的 className
    props.duration number

    过渡动画持续时间
    props.children PropTypes.node

    子组件

    Return:

    PropTypes.element

    React element

    Example:

    function Dialog(props) {
  let {show, duration} = props;

  return (
    <TranstionShow show={show} transitionName='dialog'>
      <div>啥班</div>
    </TransitionShow>
  );
}

    public classNames(args: string | object): string source 

    import {classNames} from 'daifee-react-component/src/utils.js'

    包装返回 className

    Params:

    NameTypeAttributeDescription
    args string | object

    要被包装的 className 所有 className 都回被加上项目的 namespace 前缀。 如果参数是对象，则其属性是 class 名，值为 true 则该 class 名被返回。 如果参数的属性是 _user 则直接返回其值，不作包装

    Return:

    string

    一个或多个 class。例如："class1 class2 class3"

    Example:

    let classes = classNames('dialog', {
  'dialog-btn': false,
  'dialog-title': true,
  '_user': 'user-class'
});
console.log(classes);
// 假设 namespace='rd'
// output: "rc-dialog rc-dialog-title user-class"
